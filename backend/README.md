# SpendSmart
SpendSmart is a smart budgeting app that helps you track expenses, save money, and make better financial decisions.

Description:
SpendSmart is a web-based expense tracking application designed to help users easily record, manage, and analyze their daily spending. Built with the PostgreSQL–Express–React–CSS stack, it allows users to add expenses with details such as amount, category, and date, and view them in a clean, organized dashboard.
MVP Features
1. Add Expenses: Users can add new expenses with amount, category, and date.
2. View Expenses: Users can see a list of all recorded expenses.
3. Edit and Delete: Modify or remove existing expenses.
4. Category Tracking: Group expenses by categories (e.g., Food, Transport).
5. Search Function: Quickly find specific expenses.
6. Total Summary: Display total spending and category-wise summaries.
7. Responsive Design: Works well on both desktop and mobile devices.
