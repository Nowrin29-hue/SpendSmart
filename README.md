# SpendSmart
SpendSmart is a smart budgeting app that helps you track expenses, save money, and make better financial decisions.

Description:
SpendSmart is a web-based expense tracking application designed to help users easily record, manage, and analyze their daily spending. Built with the PostgreSQL–Express–React–CSS stack, it allows users to add expenses with details such as amount, category, and date, and view them in a clean, organized dashboard.

Features:
Add Expenses ,View Expenses, Edit and Delete , Category Tracking ,Search Function ,Total summary,Responsive Design
